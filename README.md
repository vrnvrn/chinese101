# Chinese 101

Beginner Mandarin lessons built as standalone HTML presentations.

## Lessons

- **Lesson 1** (`lesson1.html`): Pinyin, tones, radicals, basic grammar, greetings, introductions, nationality. HSK 1 foundation.

## Features

- Scrollable slide format, arrow key navigation
- Interactive tone navigator (click through all 5 tones)
- Flashcard decks for vocabulary (flip to reveal pinyin and English)
- Line-by-line dialogue practice with reveal buttons
- Full vocabulary reference tables

## Structure

Each lesson is a single self-contained HTML file with no dependencies.

## Deployment

Hosted on Vercel. Each lesson file is served as a static page.
